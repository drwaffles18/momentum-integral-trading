# Notebooks exploratorios

Esta carpeta contiene notebooks correspondientes a etapas exploratorias y preliminares del proceso de investigación.  
Los análisis aquí incluidos documentan enfoques alternativos que fueron evaluados durante el desarrollo del proyecto, pero que no forman parte del modelo final presentado en el paper.

Estos notebooks se incluyen con fines de transparencia, documentación del proceso y referencia futura.

## Contenido

- **01_trading_signals_local.ipynb**  
  Exploración inicial de indicadores técnicos tradicionales, zonas de absorción y análisis del order book.

- **02_take_profit_tester.ipynb**  
  Evaluación de distintos esquemas de take-profit y stop-loss basados en ratios riesgo–beneficio, ATR, momentum y estructura de mercado.

- **03_momentum_model_dataset.ipynb**  
  Construcción de datasets para clasificación supervisada (BUY / HOLD / SELL) y experimentación con múltiples modelos de aprendizaje automático.

Estos notebooks no son necesarios para reproducir los resultados finales del paper.
